## üóÑÔ∏è Database Schema

### Main Tables

**users** - User accounts and authentication
```sql
- id (uuid, primary key)
- email (text, unique)
- name (text)
- role (text)
- created_at (timestamp)
```

**properties** - Property listings
```sql
- id (uuid, primary key)
- owner_id (uuid, foreign key ‚Üí users)
- address (text)
- description (text)
- rent_amount (numeric)
- status (text)
- created_at (timestamp)
```

**rentals** - Active rental agreements
```sql
- id (uuid, primary key)
- property_id (uuid, foreign key ‚Üí properties)
- tenant_id (uuid, foreign key ‚Üí users)
- start_date (date)
- end_date (date)
- rent_amount (numeric)
- status (text)
```

**payments** - Payment records
```sql
- id (uuid, primary key)
- rental_id (uuid, foreign key ‚Üí rentals)
- amount (numeric)
- paid_at (timestamp)
- transaction_hash (text)
- status (text)
```

**Row Level Security (RLS)**: Enabled on all tables for multi-tenant data isolation.
