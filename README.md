**Built for the AI Agents on Arc with USDC Hackathon**

## 🚀 Quick Start
```bash
# Clone repository
git clone https://github.com/your-username/rentflow-ai.git
cd rentflow-ai

# Install all dependencies
npm run install:all

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Start development
npm run dev
```
